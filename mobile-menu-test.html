Mobile Menu Test

Resize your browser window to mobile size (768px or less) to test the mobile menu.

The mobile menu should:

  • Show a hamburger menu button on mobile
  • Open a full-screen overlay when clicked
  • Have a close button (×) in the top-right corner
  • Close when clicking outside or pressing Escape
  • Close when clicking on a navigation link

Test Instructions:

  1. Resize browser to mobile width (<768px)
  2. Click the hamburger menu button
  3. Try clicking the close button (×)
  4. Try clicking outside the menu
  5. Try pressing the Escape key
  6. Try clicking on a navigation link
